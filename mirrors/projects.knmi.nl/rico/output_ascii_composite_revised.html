Required output for RICO SCM composite case in ASCII-format

We require 3 files with data:

  • Set A: Instantaneous profiles, each hour, starting at t=0.

  • Set B: Hourly averaged profile, starting with profiles averaged over the first hour

  • Set C: Time series of scalars. Instantaneous values should be provided here.

Naming convention and formatting of the data sets

Please adhere these requirements carefully to facilitate postprocessing.

Name: SetA_yourname, setB_yourname
(so in my case the data of set A are in one file called setA_siebesma, set B is in setB_siebesma, etc.

  • Formatting Set A:
    #Time=0:00hr setA yourname
    (1) (2) (3) .......(10)
    ...
    ...
    ...
    (2 blanc lines)
    Time=1:00hr setA yourname
    (1) (2) (3) .......(10)
    ...
    ...
    ...
    etc

  • Formatting Set B:
    #Time=0:30hr setB yourname
    (1) (2) (3) .......(12)
    ...
    ...
    ...
    (2 blanc lines)
    Time=1:30hr setB yourname
    (1) (2) (3) .......(12)
    ...
    ...
    ...
    etc

  • Formatting Set C:
    #setB yourname
    (1) (2) (3) .......(17)
    ...
    ...
    ...
    etc

Content of the data sets

Set A: Instantaneous profiles
(1)     {z} height at which a particular variable locates [m]
(2)     {u} Zonal wind [m/s]
(3)     {v} Meridional wind [m/s]
(4)     {thetal} Liquid water potential temperature [K]
(5)     {qt} Total water (vapor+liquid+ice) [g/kg]
(6)     {qs} saturation specific humidity [g/kg]
(7)     {ql} condensed water (liquid plus ice) [g/kg]
(8)     {qr} Rain water[g/kg]
(9)     {qr} cloud fraction [0-1]
(10)   {rho} density [kg/m^3]

Set B: Hourly averaged profiles
(1)     {z} height at which a particular variable locates [m]
(2)     {wthl} &thetal flux [W/m^2]
(3)     {wqt} qt flux, [W/m^2]
(4)     {uw} Zonal momentum flux [kg/(m s^2)]
(5)     {vw} Meridional momentum flux [kg/(m s^2)]
(6)     {Mf} Mass flux [kg/(m^3 s)]
(7)     {prec} Precipitation flux (positive downward) [W/m^2]
(8)     {w_up} updraft vertical velocity in the cloudy updraft [m/s]
(9)     {thl_up} updraft &thetal [K]
(10)     {qt_up} updraft total water specific humidity [g/kg]
(11)   {ql_up} updraft liquid water [g/kg]
(12)   {thv_up} updraft &thetav [K]

Set C: Instantaneous values of scalars
(If you use a timestep smaller than 60 secs give your instanteneous values once every minute)
(1)     {time} - Time [s]
(2)     {zcb} - Height of the lowest model level with a non-zero cloud cover [m]
(3)     {zmaxcfrac} - Height of the model level with largest cloud fraction [m]
(4)     {LWP} - Liquid water path [g/m^2]
(5)     {cc} - Total Cloud cover [0-1]
(6)     {shf} - surface sensible heat flux [W/m^2]
(7)     {lhf} - surface latent heat flux [W/m^2]
(8)     {smf} - surface momentum flux [m^2/s^2]
(9)     {tke} - Vertically integrated ρ·TKE [kg/s^2]
(10)   {prec_300} - Precipitation flux at 300 m [W/m^2]
(11)   {prec_500} - Precipitation flux at 500 m [W/m^2]
(12)   {prec_1250} - Precipitation flux at 1250 m [W/m^2]
(13)   {prec_srf} - Precipitation flux at the surface [W/m^2]
(14)   {Kh_300} - Eddy diffusivity at 300 m (m^2/s)
(15)   {Kh_500} - Eddy diffusivity at 500 m (m^2/s)
(16)   {Kh_1250} - Eddy diffusivity at 1250 m (m^2/s)
(17)   {M_base} - Mass flux at cloud base (kg/(m^2 s)
