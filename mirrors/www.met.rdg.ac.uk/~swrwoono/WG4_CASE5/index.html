Submitting Results

  • For the single long run experiment, A0, B0, C0 please tar up your results for each experiment in single file named experiment.modeller.results.run.tar [20 Dec 04]
  • For the SCM 48 hour integrations, please tar up your results from each set of experiments (e.g. A1-A13) in a single file named experiment.modeller.results.run.tar where experiment is either A1, B1 or C1. [20 Dec 04]
  • To submit your results please gzip the file and, either make them available to me via anonymous ftp or on a web page and e-mail me, S.J.Woolonugh@rdg.ac.uk, instructions on how to retreive them or e-mail them directly to me. [20 Dec 04]
